# Lown Hospital Index Analysis: Equity, Outcomes & Patient Experience (2025–2026)

## Overview
Analyzed data from 3,956 U.S. hospitals to examine how inclusivity relates to clinical outcomes, patient safety, and patient experience.

## Key Insights
- Only ~5% of hospitals received an inclusivity badge  
- Inclusive hospitals ranked significantly higher on average (**top ~18% vs ~27%**)  
- Clinical outcomes and patient safety were **nearly identical** between inclusive and non-inclusive hospitals  
- Identified a **patient experience gap (−0.52)** for inclusive hospitals  
- Found this gap is driven by **serving more disadvantaged populations**, not lower quality of care  
- Highlights potential bias in patient satisfaction metrics (e.g., HCAHPS)

## Dataset
- Source: Lown Institute Hospital Index (2025–2026)  
- Size: 3,956 hospitals  
- Data includes rankings, safety, outcomes, experience, and inclusivity metrics  

## Methods
- Cleaned and validated dataset using SQL (handled duplicates, missing rankings)  
- Compared hospitals with vs. without inclusivity badges  
- Analyzed performance across ranking tiers, clinical outcomes, and patient experience  
- Explored patterns by hospital type (urban/rural) and size
